# MiVet

MiVet es una aplicación web full stack orientada a la integración entre clínicas veterinarias y dueños de mascotas.

Para veterinarias, permite administrar clientes, mascotas e historias clínicas, así como gestionar turnos de manera simple y ordenada.

Para dueños de mascotas, permite sacar turnos, geolocalizar veterinarias, consultar especialidades y horarios de atención, y solicitar turnos de emergencia.

MiVet está pensada como una plataforma en crecimiento, con nuevas funcionalidades que se irán incorporando progresivamente.

---

## Funcionalidades actuales

- Gestión de clientes
- Registro de mascotas con historial clínico
- Administración de turnos
- Autenticación de usuarios
- Roles (administrador / usuario)

---

## Funcionalidades futuras (Roadmap)

- Geolocalización de veterinarias
- Solicitud de turnos de emergencia
- Búsqueda por especialidades
- Notificaciones
- Panel de administración avanzado
- Historial de turnos para dueños de mascotas
